Требуется разработать приложение для сбора и анализа мер поддержки промышленности с сайта ГИС Промышленность (gisp.gov.ru).

Задачи:
1. Собрать все меры поддержки (всего ~2500) с навигатора мер поддержки.
2. Для каждой меры получить следующие разделы, которые подгружаются динамически через JavaScript и имеют отдельные URL:
   - «Назначение» (основная страница меры)
   - «Требования» (URL: /nmp/measure/{id}/requirement)
   - «Необходимые документы» (URL: /nmp/measure/{id}/document)
   - «Порядок получения меры» (URL: /nmp/measure/{id}/stage)
3. Из раздела «НПА и другие документы» скачать все прикреплённые документы (PDF, DOC и т.п.). Проверить, ведут ли ссылки на реальные файлы; если ссылка битая или ведёт на страницу, а не на документ — зафиксировать ошибку. Если раздел НПА отсутствует — тоже ошибка.
4. Выполнить сравнение собранных текстовых данных (все четыре раздела) с текстом скачанных документов. Для сравнения использовать облачную LLM YandexGPT (модель yandexgpt-lite). Определять расхождения: информация в карточке отсутствует в документе, или ключевые факты из документа отсутствуют в карточке, или противоречия.
5. Сформировать итоговый отчёт в двух форматах: JSON и Excel. В отчёте должны быть:
   - список ошибок по мерам (отсутствие НПА, проблемы с документами)
   - список расхождений (с указанием фрагментов)
   - рекомендации по исправлению карточек
6. Всё решение должно работать в Docker-контейнерах на Windows 10. Использовать:
   - PHP 8.2 с расширениями
   - MySQL 8
   - Selenium Standalone Chrome для работы с динамическим контентом
   - n8n для оркестрации (запуск скриптов по расписанию/вручную)
   - библиотеки: Guzzle, Symfony DomCrawler, php-webdriver, PhpSpreadsheet, YandexGPT PHP SDK (tigusigalpa/yandexgpt-php)
7. Предоставить пошаговую инструкцию с нуля: установка Docker, создание проекта, настройка docker-compose, написание всех PHP-скриптов, создание БД, настройка n8n, получение API-ключа YandexGPT, тестирование.
8. Также дать оценку, стоит ли разбивать на микросервисы, и если нет — объяснить почему.

В ответе нужны полные листинги кода всех файлов, структура проекта, команды для запуска, примеры отчётов.
